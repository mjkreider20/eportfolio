Digital Video editing

I started this project by researching CSS and using templates in Dreamweaver. I found many tutorials on the Adobe website and blogs dedicated to web design. I found a site that offered free templates to use in Dreamweaver. I downloaded a template created by Templated. I changed the code to fit my needs and added elements that were required for this assignment. I wanted to get away from using a table to control my web content and start learning how to use CSS. This was a very good exercise for me. I have wanted to learn how to use CSS to control content and have continuity over all web pages. This assignment forced me to take the time to start learning the power of CSS and web design. I will use this same basic structure for my ePortfolio but change the color scheme and create a graphic for the banner.
